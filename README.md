# slk

## Installation

### Prepare a Slack app

1. Create New App from https://api.slack.com/apps
2. Attache the following scopes
    * channels: history
    * channels: read
    * groups: history
    * groups: read
    * im: history
    * im: read
    * mpim: history
    * mpim: read
3. Install the App to your workspaces
4. Get the OAuth Access Token
